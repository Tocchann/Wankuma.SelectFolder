# Wankuma.SelectFolder

## 概要

Windows デスクトップ用 .NET/.NET Framework 版 IFileOpenDialog フォルダ選択ダイアログです。

.NET 6 以上、.NET Framework 4.7.2 以上のWPF向けと、.NET Standard 2.0 版のコアライブラリの２つの形で用意しています。

Forms版は現時点では用意していません(セッションまでに作ってるかもしれない)。

## Wankuma.SelectFolder

.NET Standard 2.0 で実装されたフォルダ選択ダイアログ本体

WPF/WinForms のアーキテクチャから分離しているためどちらからも利用可能

## Wankuma.SelectFolder.WPF

WPF 用ラッパーライブラリ。.NET 6 以上、.NET Framework 4.7.2 以上向けにビルド

## サンプルプロジェクト

.NET 用と .NET Framework 用の２種類のサンプルを用意。プロジェクト構成はそれぞれWPFアプリとして新規作成した形
