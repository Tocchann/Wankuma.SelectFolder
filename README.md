# Wankuma.SelectFolder

## 概要

Windows Vista から追加された新しいファイルダイアログ形式によるフォルダ選択ダイアログの実装

## Wankuma.SelectFolder

.NET Standard 2.0 で実装されたフォルダ選択ダイアログ本体

WinForms からの利用も可能なように Desktop フレームワークアーキテクチャから分離

## Wankuma.SelectFolder.WPF

WPF からの利用向けに Window を設定できるようにしたサンプル

.NET8.0-windows と、.NET Framework 4.8 向けのそれぞれのライブラリをビルドする

## Samples.WPF.NetCore

.NET 製サンプルプロジェクト

## Samples.WPF.Netfx

.NET Framework 製サンプルプロジェクト
